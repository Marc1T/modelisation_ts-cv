Architecture du projet

Le projet est organisé en modules :

  • interface : Interface utilisateur (PyQt5).
  • core : Traitement des gestes et contrôle du curseur.
  • utils : Calibrage et lissage.
  • chatbot : Assistance utilisateur via FAQ.
